# The Farmer Was Replaced - Mods Collection

BepInEx mods for [The Farmer Was Replaced](https://store.steampowered.com/app/2060160/The_Farmer_Was_Replaced)

## Installation

1. Install [BepInEx 5.x](https://github.com/BepInEx/BepInEx/releases)
2. Run the game once to generate BepInEx folders
3. Copy mod DLL files to `BepInEx/plugins/`
4. Launch the game

## Mods

### [BetterLeaderboards](./BetterLeaderboards)
Enhanced leaderboard tracking and statistics

### [BetterSimulations](./BetterSimulations)
Dynamic simulation speed control with `set_simulation_speed()` function

### [Debug3](./Debug3)
Debug utilities and tools

### [NightSky](./NightSky)
Visual enhancements for the game environment

### [NomorePiggy](./NomorePiggy)
Removes the piggy bank
- Hides piggy bank GameObject
- Prevents piggy bank from collecting items
- No more cursor-following piggy bank

### [PrettyPrint](./PrettyPrint)
JSON-like formatted output with `pretty_print()` function
- No unlock required
- Proper indentation for nested structures
- Clean, readable output for dicts, lists, and sets

### [Reset](./Reset)
Progression mechanic with `reset()` function and Golden Piggy currency
- Reset game while preserving programming unlocks
- Golden Piggy currency persists across resets
- Permanent yield bonuses

### [SubtleDroneMovement](./SubtleDroneMovement)
Organic, lifelike movement for idle drones
- Unique per-drone motion patterns
- Configurable amplitude, speed, and rotation
- Performance friendly

### [UnrestrictedCanvas](./UnrestrictedCanvas)
Unrestricted canvas movement and controls
